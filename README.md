# @commenthol/microformat

> A format utility for node and browser

See <https://nodejs.org/api/util.html#util_util_format_format_args>

## Installation

```sh
npm i -S microformat
```

## Usage

```js
import format from 'microformat'
console.log(format('a %s', 'string'))
// > 'a string'
```
